# Affinne 2d transformations
## Цель:
* Отрисовать систему координат
* Отрисовать фигуру
* Реализовать трансформацию заданной фигуры путем
  умножения координ её точек на соответствующую матрицу преобразования
  * перемещение
  * отражение
  * масштабирование
  * поворот
